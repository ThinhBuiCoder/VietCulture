🗺️ Các giải pháp bản đồ thay thế Google Maps

1. Google Maps Embed API (Miễn phí)

✅ Ưu điểm:
  • Hoàn toàn miễn phí
  • Không cần API key
  • Dữ liệu chính xác
  • Dễ implement
❌ Nhược điểm:
  • Không có directions API
  • Không custom được
  • Không lấy được GPS user
  • Chỉ hiển thị tĩnh
📋 Code implementation: 
<iframe 
    src="https://www.google.com/maps/embed/v1/place?key=YOUR_API_KEY&q=${experience.location}" 
    width="100%" 
    height="400" 
    style="border:0;" 
    allowfullscreen="" 
    loading="lazy">
</iframe>

<!-- Hoặc không cần API key: -->
<iframe 
    src="https://www.google.com/maps?q=${experience.location}&output=embed" 
    width="100%" 
    height="400">
</iframe>

2. OpenStreetMap + Leaflet (Khuyến nghị)

✅ Ưu điểm:
  • Hoàn toàn miễn phí
  • Không cần API key
  • Tính năng đầy đủ
  • Directions, geocoding
  • Custom markers, popups
  • Open source
❌ Nhược điểm:
  • Dữ liệu ít hơn Google
  • Satellite images kém hơn
  • Cần học thêm API
📋 Code implementation: 
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" />
<script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script>

<div id="map" style="height: 400px;"></div>

<script>
var map = L.map('map').setView([10.7769, 106.7009], 13);
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);
L.marker([10.7769, 106.7009]).addTo(map)
  .bindPopup('${experience.title}').openPopup();
</script>

3. Mapbox (Free tier: 50,000 views/tháng)

Mapbox Demo (Cần API key để hiển thị)
✅ Ưu điểm:
  • 50,000 views miễn phí/tháng
  • UI đẹp, modern
  • Performance tốt
  • Tùy chỉnh cao
❌ Nhược điểm:
  • Cần đăng ký account
  • Có giới hạn free tier
  • Phức tạp hơn Leaflet
📋 Code implementation: 
<script src="https://api.mapbox.com/mapbox-gl-js/v2.15.0/mapbox-gl.js"></script>
<link href="https://api.mapbox.com/mapbox-gl-js/v2.15.0/mapbox-gl.css" rel="stylesheet" />

mapboxgl.accessToken = 'YOUR_MAPBOX_TOKEN';
const map = new mapboxgl.Map({
    container: 'map',
    style: 'mapbox://styles/mapbox/streets-v12',
    center: [106.7009, 10.7769],
    zoom: 13
});

new mapboxgl.Marker()
    .setLngLat([106.7009, 10.7769])
    .addTo(map);

4. HERE Maps (Free tier: 250,000 views/tháng)

HERE Maps Demo (Cần API key để hiển thị)
✅ Ưu điểm:
  • 250,000 requests miễn phí/tháng
  • Dữ liệu tốt cho châu Á
  • Directions API mạnh
  • Offline maps
❌ Nhược điểm:
  • Cần đăng ký account
  • Documentation phức tạp
  • Ít phổ biến ở VN

5. Goong Maps (Việt Nam) 🇻🇳

Goong Maps Demo (Cần API key từ map.goong.io)
✅ Ưu điểm:
  • Dữ liệu Việt Nam rất tốt
  • API bằng tiếng Việt
  • Support địa phương
  • Free tier hấp dẫn
❌ Nhược điểm:
  • Chỉ tập trung VN
  • Ít tài liệu
  • API mới, ít ổn định
📋 Đăng ký Goong Maps:
  1. Vào: https://account.goong.io/
  2. Đăng ký tài khoản miễn phí
  3. Tạo API key mới
  4. Free tier: 5,000 requests/ngày

🎯 Khuyến nghị cho dự án VietCulture:

📱 Giải pháp hybrid (Kết hợp):

  1. OpenStreetMap + Leaflet làm bản đồ chính (miễn phí 100%)
  2. Google Maps Embed làm fallback khi OSM lỗi
  3. External links để mở Google Maps, Zalo Map native apps

💡 Implementation strategy:

  • ✅ Thay file hiện tại bằng experience-detail-leaflet.jsp
  • ✅ Giữ lại các external map links (Google, Zalo, Apple)
  • ✅ Thêm copy địa chỉ để user paste vào app yêu thích
  • ✅ Progressive enhancement: OSM → Static map → External links
